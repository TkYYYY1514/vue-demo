# 数据终端 - Data Terminal

> 一个专业的本地数据管理系统，采用机械风格界面设计

## ✨ 功能特性

- 🗂️ **数据档案管理** - 完整的CRUD操作
- ⭐ **核心标记系统** - 重要数据标记
- 🔍 **智能检索** - 快速定位数据记录
- 💾 **本地存储** - 数据自动保存到浏览器
- 🎨 **机械风格界面** - 专业的终端视觉设计
- 📱 **响应式设计** - 支持多设备访问

## 🛠️ 技术栈

- Vue 3 + Composition API
- Vue Router 4
- Vite 5
- 原生 CSS3
